# AI Data Analyst OS

DataBridge AI 데이터 분석 파이프라인 대시보드 세트.

## 대시보드 3종

| 파일 | 이름 | 설명 |
|------|------|------|
| `dashboard.html` | Activity Monitor | 파이프라인 7단계 진행 모니터 |
| `analysis-dashboard.html` | Interactive Analysis Dashboard | KPI·차트·인사이트 인터랙티브 분석 대시보드 |
| `runs/latest/outputs/tableau/tableau_wireframe.html` | Executive Performance Dashboard | 태블로 연동 임원용 성과 대시보드 |

## 로컬 실행

```bash
cd ~/Projects/ai-data-analyst-os
python3 -m http.server 8080
# → http://localhost:8080/dashboard.html
```
